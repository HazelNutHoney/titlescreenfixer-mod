# Titlescreen Fixer
![Enviroment](https://img.shields.io/badge/Enviroment-Client-purple)
[![Java 25](https://img.shields.io/badge/Language-Java%2025-9B599A.svg?color=orange)](https://www.oracle.com/news/announcement/oracle-releases-java-25-2025-09-16)

[![Modrinth](https://img.shields.io/modrinth/dt/UqhikNQR?color=00AF5C&label=downloads&logo=modrinth)](https://modrinth.com/mod/titlescreenfixer)
[![CurseForge](https://cf.way2muchnoise.eu/full_1592810_downloads.svg)](https://curseforge.com/minecraft/mc-mods/titlescreen-fixer)

This revises the 26.2+ titlescreen to remove the new buttons and return its old look from 26.1 and below. This mod **does** work with _[Essential Mod](https://modrinth.com/mod/essential)_ and _[Mod Menu](https://modrinth.com/mod/modmenu)_! Below you can find some images of what it looks like.
